# 0.6 New note in Single Page App

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note over browser: User writes a new note in the SPA form and clicks Save

    Note over browser: JavaScript prevents the default form submission

    Note over browser: Browser creates a new note object with content and date

    Note over browser: Browser adds the new note to the local notes list and redraws the page

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    Note over browser,server: Request body contains the new note as JSON
    Note over server: Server stores the new note
    server-->>browser: 201 Created
    deactivate server

    Note over browser: Browser stays on the same page without a redirect or full page reload
```
